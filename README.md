# Sistema de recomendação de filmes

Os sistemas de recomendação têm se mostrado extremamente úteis tanto no mercado quanto na vida real, esses sistemas são capazes de fornecer sugestões personalizadas e relevantes para os usuários, ajudando-os a descobrir novos produtos e aumentando o engajamento do cliente.

O objetivo deste sistema é inserir um título de filme e identificar os dez filmes que são mais semelhantes a esse filme.
